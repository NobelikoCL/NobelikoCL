## Proyecto Destacado / Featured Project

### [QuantTablet](https://github.com/NobelikoCL/quanttablet)

> Dashboard de trading cuantitativo full-stack con integración en tiempo real a MetaTrader 5.
> Full-stack quantitative trading dashboard with real-time MetaTrader 5 integration.

**Stack:** Django 6 · React 19 · Vite · TailwindCSS · Pandas · NumPy · DRF · SQLite3

| Feature | Descripción |
|---|---|
| Live Account Monitoring | Actualización cada 3 segundos / 3-second live updates |
| Risk Management | Protección automática de drawdown / Automated drawdown protection |
| Multi-timeframe Scanner | Detección de fractales y confluencia EMA |
| Economic Calendar | Integración con ForexFactory |
| Copy Trading | Multi-cuenta entre distintos brokers |
| Analytics | Sharpe Ratio, drawdown máximo, métricas de performance |
